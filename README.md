\# TechPulse



An end-to-end data engineering pipeline demonstrating Airflow orchestration,

a custom MCP server for data quality and metadata enrichment, GCP (BigQuery + GCS)

as the warehouse/lake, and GenAI-powered enrichment — built as a portfolio POC.
